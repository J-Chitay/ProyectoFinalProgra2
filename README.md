En nuestra version 3, ya podemos probar en una interfaz grafica para el CRUD ingreso cliente
lo que debemos de hacer es descargar este repositorio, abrirlo en nuestro entorno de trabajo, ejecutamos nuestro InventoryFinalApplication
luego levantamos React, que vamos a la terminal, con "cd cliente-app" ingresamos a la carpeta de react, luego con "npm start" levantamos nuestro localhost
y nos abrica en http://localhost:3000 donde ya podemos observar nuestro CRUD.
